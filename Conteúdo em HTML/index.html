Tags:

<!doctype html>: É uma declaração para informar ao navegador qual é a versão do HTML utilizada no arquivo.

<html>: Esta tag é o elemento básico da estrutura do HTML.

<head>: Informações que o navegador necessita

<meta>: Define qualquer informação de metadados que não podem ser definidos por outros elementos HTML

<tittle>: Define o título do documento, mostrado na barra de título de um navegador ou na aba da página

<body>: Sua finalidade é definir o corpo do documento
